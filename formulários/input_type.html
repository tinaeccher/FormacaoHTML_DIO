Todos os Tipos de Input em HTML

Abaixo estão todos os tipos de entrada de dados suportados pelo elemento <input>.

TAG Exemplo de Código Resultado Onde usar
button <input type="button" value="Clique"> Botões clicáveis usados principalmente com JavaScript.
checkbox <input type="checkbox"> Permite ao usuário selecionar zero ou mais opções.
color <input type="color"> Abre um seletor de cores nativo.
date <input type="date"> Campo para seleção de data (dia, mês e ano).
datetime-local <input type="datetime-local"> Seleção de data e hora (sem fuso horário).
email <input type="email"> Campos que devem conter um endereço de e-mail.
file <input type="file"> Seleção de arquivos para upload.
hidden <input type="hidden"> (Invisível) Dados que não devem ser vistos ou editados pelo usuário.
image <input type="image" src="btn.png"> (Botão imagem) Define uma imagem como botão de envio (submit).
month <input type="month"> Permite selecionar mês e ano.
number <input type="number"> Campo restrito a valores numéricos.
password <input type="password"> Campo onde os caracteres são mascarados.
radio <input type="radio"> Seleção de apenas uma opção dentro de um grupo.
range <input type="range"> Controle deslizante para selecionar um valor em um intervalo.
reset <input type="reset"> Botão que limpa todos os campos do formulário.
search <input type="search"> Campo de texto otimizado para pesquisas.
submit <input type="submit"> Botão que envia os dados do formulário ao servidor.
tel <input type="tel"> Campo para inserção de números de telefone.
text <input type="text"> Tipo padrão: campo de texto simples de uma linha.
time <input type="time"> Campo para seleção de hora (horas e minutos).
url <input type="url"> Campo que deve conter um endereço de site (URL).
week <input type="week"> Permite selecionar uma semana e um ano específicos.
