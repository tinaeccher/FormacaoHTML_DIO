O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar e estruturar páginas web, funcionando como o "esqueleto" que organiza conteúdos como textos, imagens e links. Baseado em tags, ele define a estrutura do documento, que navegadores como Chrome ou Firefox interpretam para exibir o site.


    Aqui estão os pontos principais sobre HTML:

  • O que significa: Linguagem de Marcação de Hipertexto, indicando que ele estrutura textos e cria links entre páginas.
  • Função: Organizar o conteúdo de forma lógica, como títulos (<h1>), parágrafos (<p>), imagens (<img>) e listas <li>.
  • Não é Linguagem de Programação: HTML é uma linguagem de marcação, o que significa que ele organiza o conteúdo, mas não possui funcionalidades dinâmicas (como lógica de programação).
  • Base da Web: É a linguagem base utilizada no desenvolvimento web, essencial para qualquer página, geralmente combinada com CSS (estilo) e JavaScript (interatividade).


O que são tags HTML?

Voltar

Tags HTML são instruções que informam ao navegador como formatar o texto.
Você pode usar tags para definir itálico, quebras de linha, objetos, bullet points e muito mais.
Essas tags estão presentes no HTML (Hypertext Markup Language) de todas as páginas da internet.

Uma tag HTML deve conter três partes:
  1. Uma tag de abertura — começando com um símbolo < div >;
  2. Conteúdo — instruções curtas para exibir o elemento na página "div"
  3. Uma tag de fechamento — terminando com um símbolo </div>;
Formando as tags como <html>, <head>, e <body> que dizem ao navegador como exibir os elementos.
