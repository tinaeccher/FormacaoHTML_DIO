O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar e estruturar páginas web, funcionando como o "esqueleto" que organiza conteúdos como textos, imagens e links. Baseado em tags, ele define a estrutura do documento, que navegadores como Chrome ou Firefox interpretam para exibir o site.


    Aqui estão os pontos principais sobre HTML:

  • O que significa: Linguagem de Marcação de Hipertexto, indicando que ele estrutura textos e cria links entre páginas.
  • Função: Organizar o conteúdo de forma lógica, como títulos (<h1>), parágrafos (<p>), imagens (<img>) e listas <li>.
  • Não é Linguagem de Programação: HTML é uma linguagem de marcação, o que significa que ele organiza o conteúdo, mas não possui funcionalidades dinâmicas (como lógica de programação).
  • Base da Web: É a linguagem base utilizada no desenvolvimento web, essencial para qualquer página, geralmente combinada com CSS (estilo) e JavaScript (interatividade).


O que são tags HTML?

Voltar

Tags HTML são instruções que informam ao navegador como formatar o texto.
Você pode usar tags para definir itálico, quebras de linha, objetos, bullet points e muito mais.
Essas tags estão presentes no HTML (Hypertext Markup Language) de todas as páginas da internet.

Uma tag HTML deve conter três partes:
  1. Uma tag de abertura — começando com um símbolo < div >;
  2. Conteúdo — instruções curtas para exibir o elemento na página "div"
  3. Uma tag de fechamento — terminando com um símbolo </div>;
Formando as tags como <html>, <head>, e <body> que dizem ao navegador como exibir os elementos.


Qual é a diferença entre elementos, atributos e tags HTML?

Voltar

Um elemento HTML é um item da página; faz parte do conteúdo dela.
Uma tag HTML altera a maneira esse elemento aparece.
Já um atributo HTML descreve as características desse elemento.

Uma tag HTML deve conter três partes:
  1. Tags HTML contêm instruções para exibir um elemento na página. Elas começam com um < e terminam com um >. Por exemplo: <b> Marca em negrito < /b >
  2. Elementos HTML são o conteúdo da página inserido nas tags. Onde Por exemplo: <b> Marca em negrito < /b > fica igual a Marca em negrito sem as tags
  3. Atributos HTML são as propriedades sobre os elementos e aparecem na tag. <html lang="pt-BR">;

Quais são as Tags estruturais HTML?

Estas tags definem a arquitetura da página e ajudam na semântica e SEO.

TAG Exemplo Resultado Onde usar
<html> <html lang="pt-br"> (Raiz do site) A tag principal que envolve todo o código da página.
<body> <body>...</body> (Corpo da página) Onde fica todo o conteúdo visível (textos, imagens, botões).
<header> <header>Logotipo</header> (Cabeçalho) Para o topo do site ou o topo de uma seção específica.
<nav> <nav>Links de menu</nav> (Navegação) Agrupa os links principais de navegação (menus).
<section> <section>Conteúdo</section> (Seção) Divide o conteúdo em blocos temáticos (ex: "Sobre", "Serviços").
<div> <div>Container</div> (Divisão genérica) Usada para agrupar elementos para fins de design (CSS), sem valor semântico.

Quais as TAGS de Tabelas e tópicos em HTML?

As tabelas em HTML permitem organizar dados em linhas e colunas de forma estruturada.

TAG Exemplo Resultado Onde usar
<table> <table>...</table> (O contentor pai) Tag principal que envolve todos os elementos de uma tabela.
<tr> <tr>...</tr> (Uma linha) Define uma linha (Table Row) dentro da tabela.
<th> <th>Título</th> Título Define uma célula de cabeçalho (Table Header), o texto fica em negrito.
<td> <td>Dado</td> Dado Define uma célula comum de dados (Table Data).
<thead> <thead>...</thead> (Cabeçalho) Agrupa o conteúdo do cabeçalho da tabela para melhor organização.
